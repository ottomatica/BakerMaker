# Baker for Mac

### Preqs

* docker
* gtar
* hyperkit

### Using Baker for Mac

Build initramfs filesystem (places in /tmp/file.img.gz)

```
./extract-fs.sh
```

Running vm using hyperkit/vpnkit:

```
./bakerformac.sh
```
